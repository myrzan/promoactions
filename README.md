# Promoactions
<h4>Требования</h4>
<ul>
  <li>PHP 5.3></li>
  <li>СУБД MySQL</li>
</ul>
<h4>Инструкции</h4>
<ol>
  <li>Прописать в классе DatabaseAPI настройки базы данных;</li>

  <li>Запустить script.php.</li>
</ol>
